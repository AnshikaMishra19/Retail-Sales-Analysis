# 📊 Retail Sales Analysis

## 📌 Project Overview

This project focuses on analyzing retail sales data using **SQL, Python, and Power BI**. The objective is to identify sales trends, customer behavior, product performance, and business insights through data analysis and interactive dashboards.

---

## 🛠️ Tools & Technologies

- 📄 Excel
- 🗄️ MySQL
- 🐍 Python (Pandas, Matplotlib)
- 📊 Power BI
- 🌐 Git & GitHub

---

## 📂 Dataset Information

- **Dataset:** Sample Superstore
- **Rows:** 9,994
- **Columns:** 21

---

## 📊 SQL Analysis

The SQL analysis includes:

- Total Sales Analysis
- Total Profit Analysis
- Sales by Category
- Sales by Region
- Top 10 Products
- Top 10 Customers
- Top 10 States
- Advanced SQL Queries using GROUP BY, ORDER BY, HAVING, LIMIT and Aggregate Functions

---

## 🐍 Python Exploratory Data Analysis (EDA)

Performed data analysis using Python:

- Data Loading
- Data Cleaning
- Missing Value Check
- Duplicate Value Check
- Summary Statistics
- Sales by Category
- Profit by Category
- Sales by Region
- Top States by Sales
- Sales Distribution
- Data Visualization using Matplotlib

## 💡 Key Business Insights

- Technology category generated the highest sales.
- The West region achieved the highest overall revenue.
- A small number of products contributed significantly to total sales.
- Monthly sales showed fluctuations throughout the year.
- Interactive dashboard filters help users analyze sales performance efficiently.
